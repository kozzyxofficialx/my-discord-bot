📖 Overview

This Privacy Policy explains what data is collected by this Discord bot, why it is collected, how long it is retained, and your rights regarding that data.

By adding this bot to your server or using any of its commands, you agree to the collection and use of data as described in this policy.

🗃️ Data We Collect

The bot stores only what is necessary to provide its features. Below is a full breakdown:

Data What is stored Purpose
User ID Discord snowflake ID Identify users across all features

🤖 AI Features

The /ask, /imagine, and /translate commands use third-party AI APIs (Anthropic / Claude). When you use these commands:

  • Your message content is sent to the AI provider to generate a response.
  • Conversation history is stored locally in our database to maintain context.
  • You can delete your AI conversation history at any time using /clear_memory.
  • AI providers may have their own data retention policies. See Anthropic's Privacy Policy for details.

🔒 How We Use Your Data

  • To operate and deliver the bot's features as described above.
  • To allow server administrators to moderate their communities.
  • Data is never sold, shared with advertisers, or used for any commercial purpose.
  • Data is not shared with any third party except AI providers when you explicitly use AI commands.

📦 Data Storage

All data is stored in a local SQLite database on the server hosting this bot. It is not stored in any cloud database or shared storage. Access is limited to the bot process and the server dashboard.

Server settings and moderation data persist as long as the bot remains in your server. Personal user data (reminders, AFK, to-dos, AI history) persists until manually deleted or the bot is removed.

🗑️ Data Deletion

You can request deletion of your data at any time:

  • Use /clear_memory to delete your AI conversation history.
  • AFK status is automatically removed when you send a message.
  • Reminders are automatically deleted after they fire.
  • For full data deletion, contact the bot owner using the information below.

Server administrators can request full removal of their server's data by removing the bot from their server and contacting the bot owner.

👶 Children's Privacy

This bot is not directed at children under the age of 13. We do not knowingly collect data from children under 13. If you believe a child has provided data, please contact us immediately so it can be removed.
